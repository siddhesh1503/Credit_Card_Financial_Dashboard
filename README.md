# 💳 Credit Card Analytics Dashboard using Power BI & PostgreSQL

## 📖 Project Overview

The Credit Card Analytics Dashboard is an end-to-end Business Intelligence project developed using **Power BI** and **PostgreSQL**. The project provides comprehensive insights into customer demographics, credit card usage, transaction behavior, revenue generation, and interest earnings.

The solution enables stakeholders to monitor key business metrics, identify profitable customer segments, analyze spending trends, and make data-driven decisions through interactive visualizations.

The project consists of two analytical dashboards:

- **Credit Card Customer Report**
- **Credit Card Transaction Report**

---

## 🎯 Business Objectives

- Analyze customer demographics and behavior.
- Monitor revenue and interest earnings.
- Identify high-value customer segments.
- Track quarterly transaction performance.
- Understand spending patterns across expenditure categories.
- Compare performance across card categories.
- Support strategic business decision-making through data analytics.

# ⚙️ PostgreSQL Integration

The project uses PostgreSQL as the backend database for storing and managing customer, card, and transaction records.

### Workflow

1. Import CSV datasets into PostgreSQL.
2. Create normalized tables.
3. Perform SQL-based data validation and cleaning.
4. Connect PostgreSQL to Power BI using PostgreSQL Connector.
5. Transform data using Power Query.
6. Create relationships and DAX measures.
7. Build interactive dashboards.
